Tentang Saya

Halo! Nama saya Bima Satria Putra kalian bisa panggil saya Bima. Saya lahir pada tanggal 15 Maret 2010. Saat ini saya bersekolah di sekolah impian saya yaitu SMKN 1 Surabaya di kelas 10 RPL 1 . Saya sangat tertarik dengan dunia teknologi dan juga dunia programming, itulah mengapa saya memilih untuk masuk di jurusan Rekayasa Perangkat Lunak ini. Cita-cita saya adalah menjadi seorang Programmer/Software Engineer. Harapan saya untuk sekolah ini adalah semoga dapat membimbing saya untuk melangkah lebih dekat dengan cita-cita yang saya impikan.

Sebenarnya saya sudah tertarik di dunia programming sejak kelas 5 SD. Disitu saya mulai mencoba membuat sebuah server dari game minecraft hingga direkrut oleh salah satu server minecraft di Indonesia. Tapi sayangnya tidak saya lanjutkan dikarenakan ada beberapa hal yang membuat saya tidak bisa melanjutkan lagi dengan dunia programming. Dan pada akhirnya saya mulai belajar dasar-dasar pemrograman saat saya kelas 9 SMP sampai sekarang.
